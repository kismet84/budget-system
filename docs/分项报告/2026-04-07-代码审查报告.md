### 2. 路由文件内容拼接错误

`routers/ai_search.py` 文件末尾混入了 `quota.py` 和 `search.py` 的完整代码：

```
ai_search.py 末尾（约第130行后）出现了：
├── routers/quota.py 的全部代码（import + 路由定义）
└── routers/search.py 的全部代码（import + 路由定义）
```

### 6. import_quota_db.py 字段映射错误

脚本假设 JSON 数据含 `费用` 字段，但 `parse_quota_excel.py` 实际输出的字段是：`管理费`、`增值税`，没有独立的 `费用` 字段。

```python
# 当前代码（错误）
cur.execute("""
    INSERT INTO quotas (..., fees, source_file)
    VALUES (..., %s, %s)
""", (..., r.get('费用'), ...))  # ← 字段不存在

# JSON 实际字段
{
    "全费用", "其中人工费", "材料费", "机械费", "管理费", "增值税"
    # 没有 "费用" 字段
}
```

**修复方案**：移除 `fees` 字段，或在入库时计算 `fees = 全费用 - 人工费 - 材料费 - 机械费 - 增值税`。
