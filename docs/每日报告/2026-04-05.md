### 2. 数据差异分析

**发现**：PDF（1256条）vs .dat索引（1441条）存在 **217 条差异**

| 原因 | 结论 |
|------|------|
| PDF 使用 **3位编号**（E1-1），.dat 使用 **4位编号**（E1-6325） | 两个不同版本体系 |
| PDF 缺失的编号在 PDF 发布前已被删除/合并 | **217条无法补救** |
| .dat 清单 ≠ PDF 正式发布版 | PDF 已是最大完整数据 |

**结论**：`园林绿化定额2024.json`（1256条）是 PDF 能提供的全部数据，已足够用于 MVP
