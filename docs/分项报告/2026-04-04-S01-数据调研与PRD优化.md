- **园林绿化定额**：2018 版（已有，可直接采集）

## 3. PRD 重大变更（v0.4 → v0.5）

### 3.1 变更原因

用户确认：**所有数据均手动维护**，不开发爬虫。

### 3.2 主要变更

| 变更项 | v0.4 | v0.5 |
|--------|------|------|
| 数据采集方式 | 爬虫自动抓取 | ❌ 删除，改为**手动 Excel 导入** |
| 管理员职责 | 配置爬虫任务 | 每月两次手动上传 Excel |
| 技术栈 | Playwright + httpx | 已移除 |
| 管理员界面 | 无 | 新增**数据导入模块** |
| Sprint 1 任务 | 爬虫开发 8h | 替换为 Excel 导入模块 6h |

### 3.3 数据维护流程（v0.5）

```
每月两次（15日/月末）
    │
    ▼
从 bzde.hbcic.net.cn 下载 Excel
    │
    ▼
管理员登录系统 → 上传 Excel → 系统解析 → 预览确认 → 入库
    │
    ▼
写入 import_logs（记录来源/数量/时间）
```
